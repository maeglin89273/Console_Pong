環境與編譯：
執行環境在Ubuntu 14.04 LTS
Compile要加-lrt -lcurses -lpthread
小提醒：Ubuntu預設是沒有裝curses的library，需另外安裝libncurses5-dev (下sudo apt-get install libncurses5-dev)
若執行或編譯上有問題，只好煩請助教再約時間demo了

遊戲操作：
操作方式如題目規定(小寫輸入)，A、Z為左玩家的上下鍵，M、K為右玩家的上下鍵。R為減速，T為加速。Q結束遊戲。
左上角顯示的是球的速度。為了讓遊戲更有趣味性，球的行走斜率是非固定的(非只有45度角)，因此，球的移動有時抖動是正常現象。
當球拍移動時打到球，球的反射方向會因為球拍移動而影響。

程式碼注意事項：
和題目要求小小不同的是，繪圖和位置計算是由子執行緒負責，讀取鍵盤是由主執行緒負責。這樣的設計是為了讓程式碼較為簡潔。
